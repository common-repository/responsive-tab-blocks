A responsive & clean way to display your content. Create new tabs nd blocks as per your need in no-time custom type and copy-paste the shortcode into any post/page.

== Description ==
This plugin adds a "Tab Block" section to the admin panel which allows you to create tabbed content for your website the easy way. You can quickly add your content to your different tabs, choose a color, re-order them at anytime and display them anywhere with a simple shortcode. What you see is what you get, and it’s totally free.

= Responisve Tab Blocks Shortcode =

> `[grtb id=xx]`

= Available fields =
* Tab Title
* Tab Icon
** Block Title
** Block Icon
** Readmore link
** Block Content (text/images/anything)

= Available settings =
* Breakpoint (decide when the tabs should shrink into an accordion)
* Styles

== Installation ==

= Installation =
1. In your WordPress admin panel, go to Plugins > New Plugin
2. Find our Responsive Tabs plugin by WP Darko and click Install now
3. Alternatively, download the plugin and upload the contents of great-resposive-tab-blocks.zip to your plugins directory, which usually is /wp-content/plugins/
4. Activate the plugin
5. Enjoy
